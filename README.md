# secure-coding
# Secure Coding Study

안녕하세요.
이 저장소는 보안에 관심을 가지고 공부하며, 앞으로 시큐어 코딩과 웹 보안에 대해 학습한 내용을 정리하기 위한 공간입니다.

## About

앞으로 이 저장소에서는 안전한 소프트웨어 개발을 목표로, 보안 취약점의 원리와 방어 방법을 학습하고 정리할 예정입니다.
특히 OWASP에서 제시하는 주요 웹 보안 취약점을 중심으로, 실제 개발 과정에서 주의해야 할 보안 요소들을 하나씩 공부해 나가겠습니다.

## Study Topics

* OWASP Top 10
* Secure Coding
* Web Security
* Input Validation
* Authentication & Authorization
* SQL Injection
* XSS
* CSRF
* Security Misconfiguration

## Goal

단순히 기능이 동작하는 코드를 작성하는 것에서 끝나는 것이 아니라,
사용자와 시스템을 안전하게 보호할 수 있는 코드를 작성하는 것이 목표입니다.

앞으로 시큐어 코딩을 꾸준히 학습하며,
보안에 강한 개발자가 되기 위해 기록하고 성장해 나가겠습니다.
