My Secure Web Service

안전한 웹 서비스를 만들기 위한 첫 번째 프로젝트입니다.

서비스 소개

이 웹 서비스는 사용자가 편리하게 사용할 수 있으면서도, 보안적으로 안전한 구조를 갖춘 서비스를 목표로 개발할 예정입니다.

주요 기능

  • 회원가입 및 로그인 기능
  • 게시글 작성, 수정, 삭제 기능
  • 사용자 권한 관리
  • 입력값 검증
  • 데이터 보호

보안 목표

OWASP Top 10을 참고하여 SQL Injection, XSS, 인증 및 인가 취약점, 민감 정보 노출 등의 보안 문제를 예방하는 시큐어 코딩을 적용할 예정입니다.

앞으로의 계획

단순히 동작하는 웹 서비스를 만드는 것에서 끝나는 것이 아니라, 보안 관점에서 안전한 웹 서비스를 직접 설계하고 구현하는 것이 목표입니다.
